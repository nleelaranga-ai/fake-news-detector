# Fake News Detection for Regional Languages

## Problem
Fake news spreads rapidly through social media and messaging platforms, especially in regional languages,
making it difficult for users to verify authenticity.

## Solution
A web-based AI prototype that allows users to paste news or messages in regional languages
and receive a credibility classification.
